Expressão Funcional ->
    Expressões relacionais que representam a função.
    f : R -> R
        X -> f(x) (imagem) -> Aqui que entra a fórmula da função.

    Exemplo:
        f(x) = 2x+1 -> indica a fórmula para cada valor de 'x', essa é a espressão funcional.
            f(1) = (2.1)+1 = 3; f(2) = (2.2)+1 = 5;....
            No diagrama de vem, estamos conectando o valor de 1 -> 3 e 2 -> 5;

        f(x) = x²+4 ->
            f(0) = 0²+4 = 4; f(2) = 2²+4 = 8; 
            No diagrama de vem, estamos conectando o valor de 0 -> 4 e 2 -> 8;

Gráfico de uma função ->
    Representação gráfico no eixo-cartesiano onde x é o eixo horizontal e f(x) o eixo vertical;
    f(x) = 2x+1 -> representa uma reta -> Funções Injetora;
    f(x) = x²+4 -> uma parábola U por x ser positivo -> Função Sobrejetora;
    f(x) = x³   -> função cúbica -> comportamento de S sempre de crescimento;
    f(x) = sin(x) -> funções trigonométricas -> ondulação periódica;

Paridade de funções ->
    Propriedade Algébrica
    onde f(-x) = f(x) -> par
        Quando o oposto de um valor o mesmo ponto;
        Exemplo: funções quadráticas, funções cosseno;
    
    onde f(-x) = -f(x) -> impar
        Quando pegamos o oposto de x, o valor na função é o oposto;
        Exemplo: função cúbica, funções seno;
